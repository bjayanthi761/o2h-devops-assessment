# Linux Commands

## Display Current User
whoami

## Show Disk Usage
df -h

## Show Memory Usage
free -m

## Find All Log Files
find . -name "*.log"

## Create Backup Folder
mkdir backup

## Compress Directory
tar -czvf backup.tar.gz project/

## Current Directory
pwd

## List Files
ls -la

## Running Processes
ps -ef

## Network Information
ip a
